# lead-gen-automation
Overview

This project is an automated system that:

Scrapes targeted leads (niche + geo)

Filters and scores them using AI

Sends personalized DM messages

Tracks results in Google Sheets

It is built as an end-to-end pipeline using APIs and automation tools.

- Features
- Lead Scraping

Source: hashtags, keywords, bios

Data collected:

Username

Bio

Followers

Engagement

Recent post

- Lead Filtering

Followers: 200–10k

Engagement > 2%

Active users only

- AI Lead Scoring

Uses Groq API

Scores based on:

Relevance

Activity

Reply potential

- Only leads with score > 70 are selected
